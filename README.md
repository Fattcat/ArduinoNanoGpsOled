# ArduinoNanoGpsOled
Arduino Nano Gps NEO 6M Module with Oled 0.96" Display

# Connection Schematic
0.96" Display --> Arduino Nano
- VCC --> 5V
- GND --> GND
- SCL --> A5
- SDA --> A4

GPS NEO 6M Module --> Arduino Nano
- VCC --> 5V
- GND --> GND
- TX --> D5
- RX --> D6
